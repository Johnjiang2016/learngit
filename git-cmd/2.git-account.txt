关联远程仓库

使用HTTPS链接：

    码云 账号密码
    git remote add gitee-origin https://jiangchengbao:JCB58772018@gitee.com/jiangchengbao/learngit.git

    github 账号密码
    git remote add github-origin https://Johnjiang2016:jcb5877@github.com/Johnjiang2016/learngit.git

使用SSH链接：

    码云 账号密码
    git remote add gitee-origin git@gitee.com:jiangchengbao/learngit.git

    github 账号密码
    git remote add github-origin git@github.com:Johnjiang2016/learngit.git



删除本地账号配置
git remote rm github-origin  
git remote rm gitee-origin 

关联后，使用命令git push -u origin master第一次推送master分支的所有内容
git push -u origin master
(git push -u origin master 如果当前分支与多个主机存在追踪关系，则可以使用 -u 参数指定一个默认主机，这样后面就可以不加任何参数使用git push)
