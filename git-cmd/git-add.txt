1、创建一个测试文件
    touch git-readme.txt
2、使用命令  将文件添加到暂存区
    [git add <file>]将新增的文件，提交到暂存区 
    [git add . ]将修改的文件和新增的文件(不包括删除的文件)，提交到暂存区
    [git add -u] -u是 --update的缩写，将修改的文件和删除的文件提交到暂存区
    [git add -A] -u是--all的缩写，提交所用变化，即上面两个功能的合集
3、使用命令提交到本地git仓库 git commit -m <message>
    git commit -m "提交的注释说明"
